# 📺 Painel Eletrônico

Sistema em **PHP puro (8.1/8.2)** desenvolvido para a Escola Bahiana de Medicina e Saúde Pública, com o objetivo de exibir informações institucionais em televisores dos campi.

---

## 🎯 Objetivo

Exibir automaticamente:

- Aulas programadas  
- Eventos institucionais  
- Reservas de salas  
- Publicidades  

---

## 🏫 Campi

- Brotas  
- Cabula  

Cada painel exibe apenas dados do seu campus.

---

## ⚙️ Tecnologias

- PHP 8.1 / 8.2  
- HTML5  
- CSS3  
- JavaScript  
- SQL Server  
- Apache (XAMPP / Laragon)

---

## 🗄️ Dados

Os dados vêm diretamente do sistema **RESERVM**, responsável pelo cadastro de:

- Reservas de espaços   
- Publicidades  

---

## 🔄 Funcionamento

- Consulta ao banco SQL Server
- Filtragem por campus
- Organização por data e horário
- Exibição em tela cheia (modo painel)
- Atualização automática
